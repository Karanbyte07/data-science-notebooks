What is a Flash Message?

A flash message is a temporary message stored in the user's session. It is used to provide feedback after an action, appearing on the next page load and then disappearing automatically.

Steps to Use Flash Messages:

  1. Define a secret_key in your Flask application to enable session handling.
  2. Import the flash function and call it inside your view function with the message.
  3. Use get_flashed_messages() inside your HTML template to retrieve and display the messages.
